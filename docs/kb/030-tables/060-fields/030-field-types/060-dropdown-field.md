

## Drop-down list field

**In a drop-down list field, users can select a value from a drop-down list of values.**&nbsp;

## Set up a drop-down list field

When you add a new field you need to:

1. Select **Drop-down list** as your **Field type**.
2. Set the order in which you want your values to appear by selecting an option from the drop-down list under&nbsp;**Order your drop-down values**.**&nbsp;**&nbsp;
3. Click on the **Add**button under **Add drop-down values&nbsp;**to add values to the list.
4. Type your first value into the text box, and click the **Add**button again. Continue until you have added all your values.&nbsp;
