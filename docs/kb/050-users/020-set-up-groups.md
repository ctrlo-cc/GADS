# Set up a new user group


1.	Click the Admin tab in the top navigation.
2.	Select Manage Groups from the dropdown menu.
3.	Click the Create new group button
4.	Give the group a name and click Save.

NOTE: There are no system-wide field permissions associated with groups. You need to specify the permissions that each group has for each data field. You can see an overview of a groupâ€™s permissions across all fields by going to Ad- min > Manage Groups
